## 日常使用合集

### 切换 JDK 版本

#### CentOS ^[1][2]^

```sh
alternatives --config java
alternatives --config javac
```

#### Ubuntu

```shell
sudo update-alternatives --config java
sudo update-alternatives --config javac
```
