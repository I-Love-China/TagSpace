### Prerequisites
#### 安卓端
1. Android 5.0+
2. 开启 ADB 调试
#### PC 端
1. adb
2. fastboot

```shell
sudo apt install android-tools-adb android-tools-fastboot
```


### 安装
```shell
sudo snap install scrcpy
```

### 使用
1. 使用数据线连接手机
2. 命令行运行  scrcpy
