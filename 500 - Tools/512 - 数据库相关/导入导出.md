### 进度条
##### 导出时显示进度 ^[1]^
```
# 注意是 mysqlpump，不是 mysqldump
mysqlpump -h 192.168.1.21 -uroot -p --databases $DB | sed -e '/^USE `/d' > $DB.$DATE_TIME.db 
```

##### 导入时显示进度条 ^[2]^
```shell
pv sqlfile.sql | mysql -uroot -p dbname
```
