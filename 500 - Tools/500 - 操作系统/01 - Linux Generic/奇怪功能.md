```shell
mplayer -really-quiet -vo caca fireplace.mp4
```

### 时间
#### 自动获取网络时间 【开启/关闭】 ^[2]^
```shell
sudo timedatectl set-ntp true
```

#### 时区列表
```shell
timedatectl list-timezones
```

#### 设置时区
```shell
sudo timedatectl set-timezone Asia/Kolkata
```
