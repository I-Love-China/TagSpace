## 常用操作

### 设置淘宝镜像 ^[1]^
#### 临时使用淘宝镜像
```shell
npm --registry https://registry.npm.taobao.org install express
```

#### 持久使用淘宝镜像
```shell
npm config set registry https://registry.npm.taobao.org
```

#### 恢复官网镜像
```shell
npm config set registry https://registry.npmjs.org
```

### 查看全局包安装位置 ^[2]^
```shell
npm list -g --depth=0
```



### 清除缓存 ^[3]^

```bash
npm cache clean
```
