## Ubuntu18 安装L2TP/IPSec VPN 客户端



```bash
sudo add-apt-repository ppa:nm-l2tp/network-manager-l2tp
sudo apt update
sudo apt install network-manager-l2tp  network-manager-l2tp-gnome
```
