## 内存相关

### 查看进程占用的 Swap 空间大小 ^[1]^

```shell
smem  --sort=swap -r
```



### 查看进程流量消耗 ^[2]^

```
sudo iftop -i <interface_name>
```



## 运行信息查看

### 查看进程的环境变量 ^[3]^

```sh
cat /proc/2020/environ | tr '\0' '\n'
```
