## 常用功能

### 查看文件指定行的内容 ^[1]^

```shell
# Sed
sed '900!d' file

# Awk
awk 'NR=900' file
```



### 查看文件指定行范围的内容 ^[1]^

```shell
sed -n '900,920p' file
```



### 查看进程的工作目录 ^[2]^

```shell
pwdx <PID>

lsof -p <PID> | grep cwd

readlink -e /proc/<PID>/cwd
```
